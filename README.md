# Pranav Pant Portfolio

Animated one-page portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```

## Contact Form

Set `NEXT_PUBLIC_FORMSPREE_ENDPOINT` in Vercel or in `.env.local` to enable the contact form.
